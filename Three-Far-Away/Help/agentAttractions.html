All atractions

The attractions preview page has the following layout:

  1. Button for opening the hamburger menu
  2. Button for journeys
  3. Button for reports
  4. Button for navigating to the login page
  5. Dropdown list for filtering attraction type
  6. Card with basic information of one journey
  7. Button for fast operations with journey
  8. Edit button
  9. Delete button
  10. Button for detailed preview of the selected journey
  11. Button for previewing the next page
  12. Button for creating new attraction

Here, the first 4 attractions in our assortment are listed. Each card presents one attraction and contains basic information such as the attraction's name and type. To view more detailed information and edit or delete attraction, you click the button labeled "View". If there are more than 4 attractions available, you will also see a "Next" button on the screen, which fetches the next 4 attractions.

On the side, there is a menu that allows you to switch to all journeys, reports or log out of the system.

As an agent, you have quick shortcuts for editing and deleting individual attractions. You can access them by clicking on the button marked with three dots. Additionally, in the top right corner, there is a button that takes you to the process of creating a new attraction.

Shortcut List

  • Preview all journeys: CTRL + 1
  • Preview all reports: CTRL + 3
  • Log out: CTRL + 4
  • Create new attraction: CTRL + N
