See the reports for journeys

Reports is the page which user (agent) can access to via navigation bar. The page has the following layout:

  1. Filtering reports by month
  2. Filtering reports by year
  3. Preview journey page
  4. Navigating to the previous page
  5. Navigating to the next page

On this page, users (agents) can see report cards for past journeys. Every card has basic information about some journey. If user want to have detailed information about it, he can click on View button and he will be redirected to full page preview about that specific journey. Also, users can filter reports by month or (and) year.

Shortcut List

  • Move to the previous page: Ctrl + Left
  • Move to the next page: Ctrl + Right
