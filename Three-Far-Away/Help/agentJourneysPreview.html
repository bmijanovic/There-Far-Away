Journey preview

On the detailed travel information page, you can find the following data: travel name, departure and arrival dates, price, and all attractions. You can also see a map with all attractions.

The attraction card has the following layout:

Button number one is marked as the button that leads to a more detailed view of that attraction.

At the bottom of the page, we have three buttons. Button number one is the button for listing all passengers who bought this journey, button number two is the button for editing and button number three is button for deleting journey.

On the side, there is a menu that allows you to switch to all journeys, your journeys or log out of the system.

Shortcut List

  • Buy journey: CTRL + P
  • Edit journey: CTRL + E
  • Delete journey: CTRL + D
  • Preview all journeys: CTRL + 1
  • Preview self journeys: CTRL + 2
  • Log out: CTRL + 4
