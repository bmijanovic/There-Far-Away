All journeys preview

This is the first page that the agent sees after logging in. The journeys preview page has the following layout:

  1. Button for opening the hamburger menu
  2. Button for attractions
  3. Button for reports
  4. Button for navigating to the login page
  5. Card with basic information of one journey
  6. Button for fast operations with journey
  7. Edit button
  8. Delete button
  9. Button for detailed preview of the selected journey
  10. Button for previewing the next page
  11. Button for creating new attraction

When you log in to the system, you arrive at this screen. Here, the first 4 most recent journeys in our assortment are listed. Each card presents one journey and contains basic information such as the journey's name, departure and arrival dates, and price. To view more detailed information and edit, delete or see passengers, you click the button labeled "View". If there are more than 4 journeys available, you will also see a "Next" button on the screen, which fetches the next 4 journeys.

On the side, there is a menu that allows you to switch to your journeys or log out of the system.

As an agent, on the homepage, you have quick shortcuts for editing and deleting individual journeys. You can access them by clicking on the button marked with three dots. Additionally, in the top right corner, there is a button that takes you to the process of creating a new journey.

Shortcut List

  • Preview all attractions: CTRL + 2
  • Preview all reports: CTRL + 3
  • Log out: CTRL + 4
  • Create new journey: CTRL + N
