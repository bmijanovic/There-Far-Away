Attraction preview Help

The attractions preview page has the following layout:

  1. Button for opening the hamburger menu
  2. Button for journeys
  3. Button for attractions
  4. Button for reports
  5. Button for navigating to the login page
  6. Edit button
  7. Delete button

On this screen, we see detailed information about the attraction, such as its name, type, description, an image, location on the map, and address. The agent has the ability to edit or delete by selecting the appropriate button.

On the side, there is a menu that allows you to switch to all journeys, attractions, reports or log out of the system.

Shortcut List

  • Preview all journeys: CTRL + 1
  • Preview all journeys: CTRL + 2
  • Preview all reports: CTRL + 3
  • Log out: CTRL + 4
  • Edit attraction: CTRL + E
  • Delete attraction: CTRL + D
